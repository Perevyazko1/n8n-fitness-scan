# Картинки маскота-лиса «Рыж»  (copied from source repo img/README.md)

Конвенция имён (фронт строит путь сам из ответа дашборда `avatar`):

    fox_m{M}_b{B}.png

- M = muscle_tier 0..3 (мышцы, ось тренировок): 0 дрищ → 3 качок
- B = belly_tier 0..3 (живот, ось питания): 0 плоский → 3 пузо

Итого 16 файлов: fox_m0_b0 … fox_m3_b3. Размер ~300×300, прозрачный/тёмный фон.
Стиль и промпты — в памяти Claude (project_mascot.md). Цифры серий рисует UI, не картинка.

Спецсцены (опционально, позже): fox_frozen.png, fox_lost.png, fox_win.png

---
**Translation:** Naming for the "Ryzh" fox mascot. The frontend builds the path from
the dashboard `avatar` response. `M` = muscle tier 0–3 (training axis, scrawny → buff),
`B` = belly tier 0–3 (nutrition axis, flat → belly). 16 files total, ~300×300,
transparent. Streak numbers are drawn by the UI, not baked into the image.
