РыжTelegram Mini App

Политика конфиденциальности

Обновлено: 22.06.2026

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») описывает, какие данные собирает Сервис (Telegram-бот и Mini App «Рыж»), с какой целью и как они обрабатываются и защищаются.

1.2. Используя Сервис, Пользователь даёт согласие на обработку своих данных на условиях настоящей Политики. При несогласии Пользователь обязан прекратить использование Сервиса.

1.3. Политика действует совместно с Пользовательским соглашением.

2. Какие данные мы собираем

2.1. Данные аккаунта Telegram: идентификатор пользователя (ID), имя, при наличии — username. Передаются Telegram при запуске бота/приложения.

2.2. Данные профиля, которые Пользователь вводит сам: пол, рост, вес, возраст, процент жира (необязательно), цель (похудение/поддержание/набор), уровень активности, частота тренировок, целевые показатели калорий и БЖУ.

2.3. Данные, которые Пользователь вносит в дневник: записи о приёмах пищи (описание, калории, белки/жиры/углеводы, время), тренировки и упражнения, ходьба, параметры тела, личный справочник продуктов, отметки выполнения и серии (прогресс).

2.4. Сообщения AI-ассистенту (для подписчиков PRO): текстовые и голосовые сообщения, отправленные боту, и их распознанный текст — для формирования ответа.

2.5. Технические данные: сведения, необходимые для работы Сервиса (факт и время обращений, технические журналы, данные об ошибках).

2.6. Сервис не запрашивает и не собирает специальные категории данных сверх перечисленного, а также не хранит данные банковских карт (см. раздел 6).

3. Цели обработки

  • предоставление функций приложения: расчёт дневной нормы калорий и БЖУ, ведение дневника еды и тренировок, подсчёт прогресса и серий;
  • работа AI-ассистента (для PRO): ответы на вопросы, разбор питания, планы;
  • идентификация Пользователя и сохранение его данных между сессиями;
  • приём и обработка платежей за подписку;
  • поддержка Пользователей и улучшение качества Сервиса;
  • обеспечение безопасности и предотвращение злоупотреблений.

4. Правовые основания

4.1. Обработка осуществляется на основании согласия Пользователя и необходимости исполнения условий Пользовательского соглашения.

4.2. Согласие может быть отозвано в любой момент путём прекращения использования Сервиса и обращения за удалением данных (раздел 9).

5. Передача данных третьим лицам

5.1. Для работы Сервиса данные могут обрабатываться следующими поставщиками — строго в объёме, необходимом для оказания услуги:

  • Telegram — платформа, через которую работает бот и Mini App;
  • платёжный провайдер — для приёма оплаты подписки;
  • поставщик AI-технологий — для распознавания голосовых сообщений и генерации ответов ассистента (только для функций PRO);
  • открытые базы продуктов — для поиска и подстановки данных о составе продуктов по запросу Пользователя.

5.2. Сервис не продаёт и не передаёт данные третьим лицам в рекламных целях.

5.3. Данные могут быть раскрыты по обоснованному требованию уполномоченных органов в соответствии с законодательством.

6. Платёжные данные

6.1. Оплата подписки проводится через платёжного провайдера. Реквизиты банковских карт вводятся на стороне провайдера и не передаются Сервису и не хранятся им.

6.2. Сервису может поступать информация о факте, сумме и статусе платежа для предоставления доступа и поддержки.

7. Хранение и защита

7.1. Данные хранятся на защищённых серверах; доступ к ним ограничен.

7.2. Применяются разумные организационные и технические меры защиты (контроль доступа, шифрование канала передачи). При этом абсолютная безопасность передачи данных в сети не может быть гарантирована.

8. Сроки хранения

8.1. Данные профиля и дневников хранятся, пока Пользователь пользуется Сервисом, и до момента их удаления по запросу Пользователя.

8.2. Технические журналы и данные о платежах хранятся в течение срока, необходимого для работы Сервиса и соблюдения требований законодательства.

9. Права пользователя

9.1. Пользователь вправе: получить сведения о своих данных, исправить их, ограничить обработку и потребовать удаления.

9.2. Бо́льшую часть данных Пользователь может изменить самостоятельно в приложении (профиль, записи дневника).

9.3. Для удаления всех данных или иных вопросов по обработке Пользователь может обратиться в службу поддержки.

10. Файлы cookie и трекинг

10.1. Сервис не использует рекламные cookie и не ведёт стороннего рекламного трекинга. Для работы интерфейса может использоваться локальное хранилище устройства (например, выбранная тема оформления).

11. Изменение Политики

11.1. Администрация вправе обновлять Политику. Актуальная версия всегда доступна в Сервисе. Продолжение использования Сервиса означает согласие с обновлённой Политикой.

12. Контакты

12.1. По вопросам обработки персональных данных — см. страницу «Поддержка».

Используя Сервис, Пользователь подтверждает, что ознакомлен с настоящей Политикой конфиденциальности и согласен с обработкой данных на изложенных условиях.
